# Results Directory

Stores simulation output data for analysis and reproducibility.

## Structure

```
results/
├── aggregated/           # Cross-run statistical summaries
│   ├── batch_summary.json    # Per-metric mean/std/min/max across seeds
│   └── plots/                # Aggregated comparison plots
│       ├── stability_distributions.*
│       ├── metric_correlations.*
│       └── efficiency_pareto.*
└── runs/                 # Per-run raw results (timestamped)
    └── <YYYY-MM-DD_HH-MM-SS>/
        └── run_summary.json
```

## Retention Policy

Only the **latest 10 run directories** are kept under `runs/`.
Older runs are pruned during periodic maintenance to keep the repo lean.
Aggregated summaries in `aggregated/` are retained indefinitely.

## Generated By

- `core.batch_runner.BatchRunner` → `aggregated/`
- `core.simulation_runner.run_simulation()` → `runs/<timestamp>/`
