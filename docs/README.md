# Documentation

Project documentation organized by category.

## Structure

| Directory | Contents |
|-----------|----------|
| `architecture/` | System design docs (dynamic environment, feature toggles, temporal engine) |
| `theory/` | Mathematical derivations and formulations |
| `phases/` | Per-phase summaries and development notes |
| `experiments/` | Auto-generated experiment reports (one per simulation run) |
| `changelog/` | Upgrade roadmaps and day-by-day changelogs |
| `roadmap/` | Release notes and phase progression plan |
| `metrics/` | Metric definitions and stability analysis docs |
| `references/` | IEEE paper references and external resources |
